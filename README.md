## TO-DO-LIST
**A simple and interactive To-Do List web app** where you can add, view, and remove tasks directly in your browser.

**Live Demo:** https://sharmayush06.github.io/TO-DO-LIST/

---

## About

This project is a basic **To-Do List application** built with HTML, CSS, and JavaScript. It allows users to:

- Add new tasks
- View the list of tasks
- Remove tasks when completed  
It runs entirely in the browser — no backend required!

---

## Features

- Add tasks quickly  
- View your task list  
- Delete tasks you’ve finished  
- Simple and clean UI for everyday use

---

## How To Use

Just open the **Live Demo** link above in your web browser and start adding tasks!  
No installation or setup is required.
